# T18.1 · CORRECCIÓN CRÍTICA · Cataluña SÍ tiene deducción general arrendamiento

> **🚨 IMPORTANTE para CC al ejecutar T18.1.** Antes de implementar Cataluña · LEE ESTA CORRECCIÓN.
>
> El documento `docs/T18-cifras-Top5-pre-investigadas.md` afirmaba que Cataluña NO tiene deducción general de arrendamiento. **ESTO ES INCORRECTO.**
>
> **Cataluña SÍ tiene deducción general de arrendamiento de vivienda habitual** desde el Decreto Legislativo 1/2024 (12 marzo 2024 · libro sexto Código tributario de Catalunya).

## 1 · Cataluña · Deducción por alquiler de vivienda habitual ★ CORRECTO

### 1.1 · Cifras verificadas Manual AEAT 2025

| Concepto | Valor |
|---|---|
| Porcentaje | **10%** de cantidades satisfechas en periodo impositivo |
| Tope general | **500 €** anuales |
| Tope incrementado | **1.000 €** anuales (familia numerosa o monoparental · O si en tributación conjunta uno declarante cumple requisito edad/paro) |
| Edad/perfil elegible | <ul><li>Tener **35 años o menos** a fecha devengo (31/12)</li><li>**O** haber estado en **paro 183 días o más** durante el ejercicio (Seguridad Social facilita información)</li><li>**O** pertenecer a **familia numerosa** (Ley 40/2003)</li><li>**O** **familia monoparental** (Ley 18/2003 + Decreto 151/2009)</li></ul> |
| BI máx individual (general + ahorro - mínimo personal y familiar) | TODO CC verificar contra Manual AEAT 2025 |
| BI máx conjunta | TODO CC verificar |
| Identificación arrendador | NIF obligatorio en declaración-liquidación |
| Tope por vivienda compartida | 1.000 € total · prorrateado entre titulares con derecho |
| Casilla | Anexo B.11 |
| Fuente OFICIAL | Art. 612-3 Decreto Legislativo 1/2024 (12 marzo 2024) · libro sexto Código tributario Catalunya · `https://sede.agenciatributaria.gob.es/Sede/ayuda/manuales-videos-folletos/manuales-practicos/irpf-2025-deducciones-autonomicas/comunidad-autonoma-cataluna/alquiler-vivienda-habitual.html` |

// Test 1 · Cataluña · joven elegible

// Test 4 · Cataluña · 50 años · NO paro · NO familia numerosa → NO ELEGIBLE
